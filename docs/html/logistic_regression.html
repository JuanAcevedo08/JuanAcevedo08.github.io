Regresión Logistica: Fundamentos y Aplicación

Publicado el 22 Agosto 2025 · Autor: Juan Acevedo

¿Qué es la Regresión Logistica?

es un modelo matemático que surge del intento de utilizar la regresión lineal para predecir valores, pero en lugar de obtener un valor continuo, se transforma en una probabilidad. A través de diversas herramientas matemáticas se desarrolló lo que hoy conocemos como la función sigmoide, la cual permite convertir la salida de la regresión lineal en una probabilidad entre 0 y 1, lo que nos ayuda a clasificar principalmente entre dos categorías. Este modelo puede incorporar múltiples variables; sin embargo, a menudo se ilustra con dos por simplicidad. La función de pérdida utilizada es la entropía cruzada: L(ŷ, y) = - (y · log(ŷ) + (1-y) · log(1-ŷ)), donde se mide la diferencia entre el valor real y el valor predicho dependiendo de la clase. El modelo se optimiza mediante descenso del gradiente, comenzando con valores iniciales para los parámetros (betas) y ajustándolos en cada iteración hasta que el gradiente tienda a 0, buscando así minimizar el error lo máximo posible.

Modelo Matemático

En su forma simple, el modelo se expresa como:1 / (1 + e^-z) donde z = β₀ + β₁x , donde β₀ es el intercepto, β₁ la pendiente y lo que busca es ajustar la supercicie para que den la probabilidad lo mas exacto posible y con esa probabilidad si > 0.5 = 1 de lo contrario = 0

Buenas Prácticas

  • Normalizar o estandarizar si las escalas son muy diferentes.
  • Revisar multicolinealidad .
  • Analizar residuos para validar supuestos.
  • Datos balanceados (50%, 50%).
  • Usar métricas como accuracy_score .
  • Revisar correlaciones.

Limitaciones

Asume linealidad, tiene a overfitting si tiene muchas caracteristicas, la multicolonealidad lo afecta y los datos desbalanceados
