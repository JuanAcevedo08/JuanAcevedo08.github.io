# 🚀 Portafolio Machine Learning Engineer

Un portafolio web moderno, minimalista y profesional diseñado específicamente para Machine Learning Engineers principiantes. Construido con HTML5, CSS3 y JavaScript vanilla, optimizado para reclutadores y empresas tecnológicas en 2025.

## ✨ Características Principales

### 🎨 Diseño Visual
- **Tema oscuro minimalista** con acentos neón (azul, violeta, verde)
- **Tipografía moderna** usando Inter y Poppins
- **Efectos visuales avanzados** con resplandores y animaciones suaves
- **Paleta de colores profesional** optimizada para reclutadores tech

### 🧠 Animaciones Neurales
- **Red neuronal interactiva** como fondo dinámico
- **Nodos que reaccionan al mouse** con efectos de conexión
- **Partículas animadas** que siguen el cursor
- **Efectos de pulso y resplandor** en tiempo real

### 📱 Diseño Responsivo
- **Mobile-first approach** con breakpoints optimizados
- **Navegación hamburguesa** para dispositivos móviles
- **Layouts adaptativos** para tablet, móvil y desktop
- **Rendimiento optimizado** para todas las pantallas

### ⚡ Interactividad Avanzada
- **Animaciones de scroll** con Intersection Observer
- **Efectos hover** con glow neón en tarjetas y botones
- **Barras de progreso animadas** para skills
- **Scroll suave** entre secciones
- **Tooltips informativos** en elementos interactivos

## 🎨 Personalización

### 🔧 Datos Personales
Edita el archivo `index.html` para personalizar:
- **Línea 65**: Cambia "Tu Nombre Completo"
- **Línea 66**: Actualiza el título profesional
- **Línea 67-70**: Modifica la descripción personal
- **Línea 350+**: Actualiza links de GitHub, LinkedIn y email

### 🎨 Colores y Tema
En `css/style.css` (líneas 2-20), personaliza las variables CSS:
```css
:root {
  --accent-primary: #00f5ff;    /* Cyan neón */
  --accent-secondary: #8a2be2;  /* Violeta */
  --accent-green: #00ff88;      /* Verde neón */
}
```

### 📊 Skills y Progreso
En `index.html`, actualiza las barras de progreso:
```html
<div class="skill-progress" data-progress="85"></div>
```
